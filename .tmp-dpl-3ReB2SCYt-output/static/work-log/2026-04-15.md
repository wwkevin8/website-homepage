## Key decisions
- Kept email/password as the primary auth path while preserving Google sign-in as an alternate entry point.
- Split transport request handling from group membership and lifecycle logic so admin and public endpoints can share smaller helper modules.
- Introduced separate public handler routing instead of keeping all old submission scripts in `api/public/`.
- Added explicit runtime and deployment config so local and Vercel behavior does not depend on implicit defaults.
